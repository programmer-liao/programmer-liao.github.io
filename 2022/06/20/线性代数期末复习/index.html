第一章 n阶行列式

1、逆序数

定义

在一个排列中，如果两个数（称为数对）的前后位置与大小顺序相反，即前面的数大于后面的数，那么称这个这两个数构成一个逆序。一个排序中逆序的总数称为这个排列的逆序数

求逆序数

例：求$42351$的逆序数

逆序数为$0 + 1 + 1 + 0 + 4 = 6$

2、行列式的性质

性质1

行列式的值与它的转置行列式的值相等

性质2

互换行列式的两行（列），行列式的值反号

推论1

若行列式有两行（列）元素对应相等，则该行列式的值为零

性质3

行列式的某一行（列）中所有元素都乘以同一个数k，等于用数k乘以该行列式

推论2

若行列式有两行（列）元素对应成比例，则该行列式的值为零

性质4

若行列式某行（列）的元素都是两个数的和，则行列式D等于两个行列式之和

性质5

把行列式某一行（列）的元素乘以数k，加到另一行（列）对应的元素上，行列式的值不变

3、行列式的计算

代数余子式

在行列式中划去$a_{ij}$所在的第$i$行和第$j$列，剩下的$(n-1){2}$个元素按原来的顺序排成一个$n-1$阶行列式，称为元素$a_{ij}$的余子式，记为$M_{ij}$，又记$A_{ij}=(-1){i+j}M_{ij}$，称为元素$a_{ij}$的代数余子式

定理1.3

行列式$D$等于它的任一行（列）的各元素与其对应的代数余子式乘积之和，即

$D=a_{i1}A{i1}+a_{i2}A{i2}+…+a_{in}A{in}\qquad(i=1,2,…,n)$

$D=a_{1j}A{1j}+a_{2j}A{2j}+…+a_{nj}A{nj}\qquad(j=1,2,…,n)$

4、克拉默法则

用途

求解$n$元线性方程组的解

定义

如果线性方程组的系数行列式$D$不等于零，那么方程组有唯一解

$x_1 = \frac{D_1}{D},x2= \frac{D_2}{D},…,x_n = \frac{D_n}{D}$

其中$D_j(j=1,2,…,n)$是把系数行列式D中的第$j$列元素用方程组右端的常数项代替后所得到的的$n$阶行列式

定理1.4

如果线性方程组的系数行列式$D \neq 0$，则方程组一定有解，且解是唯一的

定理$1.4^{'}$

如果线性方程组无解或有两个不同的解，则它的系数行列式必为零

定理1.5

如果齐次方程组的系数行列式不等于零，则齐次线性方程组没有非零解

定理$1.5^{'}$

如果齐次线性方程组有非零解，则齐次线性方程组的系数行列式必为零

第二章 矩阵

1、相关概念

  1. 在$m \times n$矩阵A中，如果$m=n$，则称$A$为$n$阶方阵

  2. 如果矩阵$A$的元素全为实（复）数，就称$A$为实（复）矩阵

  3. 只有一行的矩阵称为行矩阵，只有一列的矩阵称为列矩阵

  4. 当两个矩阵的行数相等，列数也相等的时，则称它们为同型矩阵

  5. 元素都是零的矩阵称为零矩阵

  6. 主对角线全为$1$，其他元素都是$0$的$n$阶方阵称为$n$阶单位矩阵

2、矩阵的运算

交换律

$A+B=B+A$

结合律

$(A+B)+C=A+(B+C)$

数与矩阵相乘

  1. $(\lambda \mu)A=\lambda(\mu A)$

  2. $(\lambda + \mu)A = \lambda{A} + \lambda{B}$

  3. $\lambda{A + B}=\lambda A + \lambda B$

矩阵的乘法

设 $A=( a _ {ij} ) _ { m \times s},B = (b _ { ij })_ {s \times {n} }$,定义矩阵$A$与$B$的乘积为$C=(c_{ij})_{m \times n}$,并把此乘积记作 $ C = A B $

矩阵的转置

定义

把矩阵$A$的行换成同序数的列，得到的新矩阵称为A的转置矩阵，记作$A^{T}$

公式

  1. $ ( A B ) ^ { T }=( B A ) ^{ T }$

对称矩阵和反对称矩阵
对称矩阵

设$A$为$n$阶方阵，若$A^{T}=A$，则称$A$为对称矩阵

反对称矩阵

设$A$为$n$阶方阵，若$A^{T}=-A$，则称$A$为反对称矩阵

方阵的行列式

由$n$阶方阵$A$的元素构成的行列式（各元素位置不变），称为方阵$A$的行列式，记作$|A|$或$detA$

3、矩阵的逆

奇异矩阵和非奇异矩阵

设$A$为$n$阶方阵，若$|A|=0$，则称$A$为奇异矩阵，否则，陈$A$为非奇异矩阵

定理2.1

设$A$是$n$阶方阵，$A$是非奇异矩阵的充要条件为$A$是可逆的

推论

如果$n$阶方阵$A,B$满足$AB=E$，则$A,B$均可逆，且

$ A ^ { -1 }= B ,B ^ { -1}=A, B A = E$

4、矩阵的分块

将矩阵$A$用若干条纵线和横线分成许多个小矩阵，每个小矩阵称为$A$的子块，以字块为元素的矩阵称为分块矩阵

第三章 向量组与向量的秩

1、相关概念

2、线性相关和线性无关

称向量组$\alpha_1,\alpha_2,…,\alpha_s$线性相关，如果有不全为零的数$k_1, k_2, …，k_s$，使得$\sum_{i= 1}^{s}{k_i\alpha_i}=k_1\alpha_1+k_2\alpha_2+…+k_s\alpha_s=0$

如果只有$k_1=k_2=…=k_s=0$式子才成立，就称$\alpha_1,\alpha_2,…,\alpha_s$线性无关

3、矩阵的初等变换

内容

  1. 对换矩阵两行（列）的位置

  2. 矩阵的某行（列）所有元素同乘以一个非零常数

  3. 把矩阵一行所有元素的$k$倍加到另一行对应的元素上去

定理

  • 如果矩阵$A$经过有限次初等行变换变为$B$,则$A$的行向量组与$B$的行向量组等价，而$A$的任意$k$个列向量与$B$中对应的$k$个列向量有相同的线性关系

  • 如果矩阵$A$和矩阵$B$等价，那么$R(A)=R(B)$

4、求矩阵的逆

利用矩阵的初等行变换求矩阵的逆

第四章 线性方程组

1、 消元法

解线性方程组最常用的方法–高斯消元法

逐步消减变元的个数，把原方程组化为等价的阶梯形方程组，再用回代过程解此等价的方程组，从而得出原方程组的解

线性方程组的初等变换

  1. 交换两个方程的位置

  2. 用一个不等于$0$的数乘某个方程

  3. 用一个数乘某一个方程加到另一个方程上

系数矩阵和增广矩阵

  • 系数矩阵

    把线性方程组的系数所组成的矩阵叫做线性方程组的系数矩阵

  • 增广矩阵

    把系数及常数项所组成的矩阵叫做增广矩阵

2、线性方程组有解判定定理

线性方程组有解的充要条件是：系数矩阵与增广矩阵有相同的秩$r$

  1. 当$r$等于方程组所含未知量个数$n$时，方程有唯一解

  2. 当$r<n$时，方程组有无穷多解

线性方程组无解的充要条件是：系数矩阵的秩与增广矩阵$\overline{A}$的秩不相等

3、线性方程组解的结构

定理4.4

一个齐次线性方程组有非零解的充要条件是：它的系数矩阵的$r$小于它的未知量的个数$n$

定理4.5

齐次线性方程组若有非零解，则它一定有基础解系，且其基础解系所含解向量的个数等于$n-r$，其中$r$是系数矩阵的秩

第五章 特征值与二次型

1、向量的内积

定理5.1

若$n$维非零向量组$\alpha_1,\alpha_2,…,\alpha_n$为正交向量组，则它们为线性无关向量组

2、方阵的特征值和特征向量

计算$n$阶方阵$A$的特征值和特征向量的步骤

  1. 计算特征多项式$|A-\lambda E|$

  2. 求出特征方程c的全部根，它们就是$A$的全部特征值

  3. 对于$A$的每一个特征值$\lambda_i$，求出相应的齐次线性方程组$(A-\lambda_i E)=0$的一个基础解系为

    $\alpha_1,\alpha_2,…,\alpha_r$

​ 则对于不全为零的任意常数$k_1\alpha_1+k_2\alpha_2+…+k_r\alpha_r$

​ 即为对应于$\lambda_i$的全部特征向量
